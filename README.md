# 💼 Бизнес помощник

Telegram-бот для напоминания о деловых встречах и задачах.

## 🚀 Возможности

- Ежедневные напоминания о задачах
- Отслеживание дедлайнов
- Приоритеты задач (высокий, средний, низкий)
- Автоматическая отправка по расписанию

## 📋 Настройка

1. Отредактируйте `tasks.json` — добавьте свои задачи
2. Настройте расписание в `.github/workflows/main.yml`
3. Добавьте секреты в GitHub Actions

## 🔧 Запуск

Бот работает автоматически по расписанию (каждый будний день в 9:00 UTC).

Также можно запустить вручную: Actions → Business Assistant → Run workflow
